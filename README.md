# Fekhar - Cerámica Artesanal Marroquí

Fekhar es una página web responsiva dedicada a la promoción y exhibición de cerámica artesanal tradicional marroquí. Este proyecto combina un diseño elegante, tradicional y minimalista con una experiencia de usuario fluida y adaptada a dispositivos móviles.

---

## Características del Proyecto

* **Diseño Mobile-First & Responsivo:** La interfaz se adapta perfectamente desde pantallas de teléfonos móviles hasta monitores de escritorio mediante CSS Grid y Flexbox.
* **Sección de Colecciones:** Galería organizada por categorías (Vajilla tradicional, Jarrones y Alfarería) con efectos visuales interactivos al pasar el cursor (hover).
* **Sección de Historia:** Integración de contenido multimedia (video en bucle con atributos `autoplay` y `muted`) que relata el origen milenario de las piezas.
* **Formulario de Contacto Validado:** Formulario funcional que procesa y extrae los datos de forma moderna utilizando `FormData` de JavaScript, con mensajes de error personalizados en tiempo real y confirmación de éxito.
* **Detalles Profesionales:** Inclusión de un menú de navegación fijo (`sticky header`), anclas con desplazamiento suave (`smooth scroll`) y un elemento visual de identidad (`Favicon`) en la pestaña del navegador.

---

## Tecnologías Utilizadas

* **HTML5:** Estructura semántica del sitio web (`header`, `main`, `section`, `footer`).
* **CSS3:** Estilos personalizados, arquitectura de diseño moderna (Flexbox y Grid Layout) y variables nativas para la gestión de colores de estado.
* **JavaScript (ES6):** Manipulación dinámica del DOM para la validación del lado del cliente sin recargar la página.
* **Git & GitHub:** Control de versiones utilizando una metodología de ramas (`develop` y `main`).
